# Defensa y Asalto de Base

Juego de estrategia por turnos para dos jugadores desarrollado en Python con Tkinter.

## Integrantes
- Francisco Li
- Emanuel Gutiérrez

## Requisitos
- Python 3.10+
- Tkinter (incluido con Python)

## Cómo ejecutar
```bash
python main.py
```

## Estructura del proyecto
```
proyecto/
├── main.py              # Punto de entrada
├── vistas/              # Ventanas Tkinter
│   └── menu.py
├── clases/              # Lógica del juego (próximamente)
└── datos/
    └── jugadores.json   # Registro de jugadores
```
